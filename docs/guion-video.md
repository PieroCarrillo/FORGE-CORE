# Guion de Video

Duracion sugerida: 5 a 7 minutos.

## 1. Introduccion

Presentar FORGE CORE como e-commerce ficticio de hardware gamer para el curso de Sistemas Operativos.

## 2. Arquitectura AWS

Mostrar el diagrama: navegador, App Server EC2, Database Server EC2 y security groups.

## 3. Demo visual

Recorrer Inicio, Catalogo, Detalle de Producto, Carrito y Admin.

## 4. Demo backend

Mostrar endpoints principales y explicar que Express se conecta a MariaDB por IP privada.

## 5. Demo base de datos

Abrir MariaDB y mostrar tablas principales. Ejecutar o explicar `sp_create_simulated_order`.

## 6. Conceptos de Sistemas Operativos

Explicar el worker thread, lectura de `/proc`, uso de `df`, metricas de CPU/RAM/disco/procesos y sincronizacion con transacciones.

## 7. Cierre

Resumir funcionamiento, seguridad y cumplimiento de la guia.

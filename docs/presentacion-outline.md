# PPT: Estructura Recomendada

## 1. Portada

FORGE CORE / PC Hardware Hub. E-commerce ficticio de componentes tecnologicos en AWS.

## 2. Problema

Necesidad de demostrar una aplicacion cloud funcional que combine e-commerce con conceptos de Sistemas Operativos.

## 3. Arquitectura

Mostrar dos EC2: App Server y Database Server. Explicar Nginx, Express, React y MariaDB.

## 4. Frontend

Hero visual, catalogo, detalle RTX 4090 Ultra, carrito de combate y panel admin.

## 5. Backend

API REST, validacion de entradas, conexion por IP privada a MariaDB y endpoints principales.

## 6. Base de datos

Tablas, relaciones, seed data y stored procedures.

## 7. Sistemas Operativos

Worker thread, `/proc`, `df`, modulo `os`, metricas del servidor y concurrencia.

## 8. Sincronizacion

Checkout con transaccion, bloqueo de filas mediante `SELECT ... FOR UPDATE` y rollback por stock insuficiente.

## 9. Seguridad

Security groups, MariaDB privado, usuario limitado, variables de entorno y Nginx.

## 10. Demo

Flujo: catalogo, carrito, pago simulado, dashboard admin y consulta de metricas.

## 11. Conclusiones

La solucion cumple la guia y muestra una aplicacion cloud completa con evidencias tecnicas.
